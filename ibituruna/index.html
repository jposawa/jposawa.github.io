Do Tupi-Guarani que significa "Nuvem Negra", Ibituruna é um grupo de estudos cuja pesquisa principal é a Computação em Nuvem, mas não se limitando somente a este tema

Aspectos de análise de desempenho, banco de dados e computação de alto desempenho também são discutidos, assim como tópicos específicos de cada área de estudo
