Antes de mais nada, é importante lembrar que a forma como as questões estão feitas no Kata não são a melhor maneira, justamente para estimular que vocês modifiquem a resolução com as melhorias que julgarem mais adequadas.

Nesse primeiro momento, as questões foram resolvidas em Portugol para abstrair o algoritmo da sintaxe de programação. De modo que seguindo esses passos em qualquer linguagem você pode conseguir reproduzir o resultado.

Bom treino.


Questão 01

Enunciado

Faça um algoritmo que o Usuário informa duas notas diferentes, de 0 a 10. A média das duas notas é calculada e baseado em seu resultado, informa se o aluno a Média e se o aluno está Reprovado, vai para Avaliação Final ou foi Aprovado.

Os critérios para os resultados são: À partir de 7, o aluno está Aprovado; À partir de 4 e menos que 7, vai para Avaliação Final; Abaixo de 4 é Reprovado.


Resolução

O primeiro passo é identificar o problema e quais os passos para resolver. Nesse caso o problema é calcular a média e informar qual a situação do aluno de acordo com ela.

Dessa maneira, primeiro o sistema recebe as duas notas, que o usuário insere, e faz os cálculos necessários, além de categorizar seguindo suas condições.

Tendo isso em mente, vamos ao algoritmo em Portugol:

Inicio

  real nota1, nota2, soma, media


  escrever "Insira a primeira nota"

  ler nota1

  escrever "Insira a segunda nota"

  ler nota2


  soma <- nota1+nota2

  media <- soma/2

  se media >= 7 entao

    escrever "Aprovado com a Média ",media

  fimse

  se media < 6 E media >= 4 entao

    escrever "Vai para a AF com a Média ",media

  fimse

  se media < 4 entao

    escrever "Reprovado com Média ",media

  fimse

Fim.

Questão 02

Enunciado

Você recebe a solicitação de desenvolver um algoritmo para calcular quanto deve ser cobrado de multa para motoristas que passam acima da velocidade máxima de um sensor.

A solicitação pede que para demonstrar seu algoritmo seja inserido no programa a velocidade limite e a velocidade de demonstração na qual o veículo hipotético teria passado.

É pedido que o programa tenha uma tolerância de 5% da velocidade do veículo para a velocidade limite. Além disso, é determinado que existem quatro níveis de infração, a serem definidos dependendo de quantos porcento (%) o veículo estava acima da velocidade limite. A cada 15%, um nível de gravidade é acrescentado, recebendo os nomes de LEVE, MODERADA, GRAVE e GRAVÍSSIMA.

O resultado desejado é a informação da velocidade, sem casas decimais, da velocidade do veículo, de qual nível de gravidade a infração se enquadra, além do preço da multa. Sabendo que ela é calculada tendo um valor base de R$ 110,00 acrescidos de R$ 25,00 para cada nível de gravidade acima de LEVE.


Resolução

Antes de ir para a codificação, é importante determinar exatamente quais dados você tem e quais informações precisa passar para o usuário.

Nesse caso sabemos que a Velocidade Máxima permitida será inserida pelo usuário, assim como a velocidade do veículo. Sabemos também que até 5% acima da Velocidade Máxima é tolerado como não havendo infração.

A cada 15% acima da velocidade, a infração sobe um nível. Então temos quatro intervalos: De 5% até 15% acima da velocidade, de 16% até 30% e de 31% até 45% e acima de 45%.


Sabendo isso, vamos ao algoritmo em Portugol:

Inicio

  inteiro vel_maxima, vel_veiculo, vel_tolerada, vel_moderada, vel_grave, vel_gravissima

  real multa

  texto situacao


  escrever "Insira a Velocidade Máxima do a via"

  ler vel_maxima

  vel_tolerada <- vel_maxima + vel_maxima * 5/100

  vel_moderada <- vel_maxima + vel_maxima * 15/100

  vel_grave <- vel_maxima + vel_maxima * 30/100

  vel_gravissima <- vel_maxima + vel_maxima * 45/100


  escrever "Insira a Velocidade do Veículo na via"

  ler vel_veiculo


  se vel_veiculo <= vel_tolerada entao

    situacao "velocidade Regular"

  se vel_veiculo > vel_tolerada entao

    multa <- 110

    situacao <- "Infração Leve. A multa a ser paga é de R$"

  fimse

  se vel_veiculo > vel_moderada entao

    multa <- multa + 25

    situacao <- "Infração Moderada. A multa a ser paga é de R$"

  fimse

  se vel_veiculo > vel_grave entao

    multa <- multa + 25

    situacao <- "Infração Grave. A multa a ser paga é de R$"

  fimse

  se vel_veiculo > vel_gravissima entao

    multa <- multa + 25

    situacao <- "Infração Gravíssima. A multa a ser paga é de R$"

  fimse


  escrever "O veículo estava a ", vel_veiculo, " o que representa ", situacao, multa

Fim.
